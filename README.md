# The Madness Club - Gastronomic Consultancy Website

This project is the official website for "The Madness Club," a gastronomic consultancy service based in Barcelona. The website aims to present the services offered, showcase results achieved with other clients, explain the working methodology, and provide contact information for potential clients.

## Website Sections

The website is a single-page application with the following main sections:

*   **Hero:** Introduction and main call to action.
*   **Locura (Madness):** A section emphasizing the passion behind gastronomic ventures.
*   **Servicios (Services):** Details on how "The Madness Club" can help.
*   **Cómo Funciona (How it Works):** The process of engagement.
*   **Resultados (Results):** Showcasing past successes and client achievements.
*   **Método (Method):** The consultancy's approach and philosophy.
*   **Quién Soy (Who I Am):** Information about the consultant.

## Technologies Used

*   **HTML:** For the structure of the website.
*   **CSS:** For styling and layout.
*   **JavaScript:** For interactive elements (e.g., mobile navigation).

## Setup and Usage

This is a static website. To view it:

1.  Clone or download the repository.
2.  Navigate to the project's root directory.
3.  Open the `index.html` file in your preferred web browser.

No special build steps or dependencies are required.
